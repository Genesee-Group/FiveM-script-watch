# FiveM-script-watch

Um script moderno e elegante para substituir a HUD padrão do FiveM, oferecendo uma experiência imersiva com apps integrados, design responsivo e atualizações em tempo real.

## ✨ Características Principais

- 🕒 **Sistema de Hora In-Game**: Relógio sincronizado com o tempo do servidor.
- ❤️ **Status do Jogador**: Vida, fome e sede atualizados em tempo real.
- 🆔 **RG Digital**: Visualização rápida de informações pessoais.
- 🎵 **Player de Música**: Integração com YouTube e Spotify, com pesquisa e indexação.
- 🗺️ **Sistema de GPS**: Navegação intuitiva no mapa.
- 📱 **Apps Integrados**: Música, GPS, Passaporte, Configurações e Atividades.
- 🎨 **Design Personalizável**: Mova o relógio e personalize a interface.
- 🔄 **Atualizações em Tempo Real**: Todos os status sincronizados instantaneamente.

## 🚀 Instalação

### Pré-requisitos
- [FiveM Server](https://fivem.net/) configurado.
- [xSound](https://github.com/Xogy/xsound) para o player de música (opcional, mas recomendado).

### Passos de Instalação
1. **Baixe o script**: Clone ou baixe este repositório.
2. **Descompacte**: Extraia os arquivos para sua pasta de recursos do FiveM.
3. **Mova para recursos**: Coloque a pasta `FiveM-script-watch` na pasta `[resources]/` do seu servidor.
4. **Configure o server.cfg**:
   ```
   ensure xsound
   ensure FiveM-script-watch
   ```
5. **Reinicie o servidor**: Execute `restart FiveM-script-watch` no console do servidor.

## ⚙️ Configuração

Personalize o script editando o arquivo `config/config.lua`. Você pode ajustar:
- Posições dos elementos na tela.
- Cores e temas.
- Funcionalidades habilitadas/desabilitadas.
- Integrações com APIs externas.

Para mais detalhes, consulte o arquivo de configuração.
